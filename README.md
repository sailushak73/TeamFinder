# TeamFinder

TeamFinder is a platform that helps students and developers find teammates for hackathons and collaborative projects. 
Users can create recruitment posts, discover skilled teammates, send join requests, manage team applications, and communicate through an integrated messaging system.

## Live Demo

Frontend:  
https://team-finder-lilac.vercel.app/

Backend API:  
https://teamfinder-l39w.onrender.com

---

## Features

- User Authentication (Register & Login)
- Create and manage recruitment posts
- Discover developers and teammates
- View detailed user profiles
- Send join requests to teams
- Accept or reject team requests
- Personal profile management
- One-to-one messaging system
- Responsive and clean user interface

## Key Functionalities

### Recruitment Posts

Users can create recruitment posts describing their project, required skills, and team requirements.

### Team Discovery

Students can browse available opportunities and explore profiles of potential teammates.

### Request Management

Users can send requests to join teams, while team owners can accept or reject applications.

### Messaging

Direct messaging allows team members to communicate and coordinate effectively.
